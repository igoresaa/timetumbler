**TimeTumbler** это просто учебный проект на C++.
TimeTumbler - CLI утилита для создания заметок и TODO

Команды:
-create: создать заметку
-list: вывести список заметок
-desc: вывести описание заметки
-rm: удалить заметку
-switch: переключить заметку на выполненную / не выполненную

**Установка:**
Просто скачайте bild из release и запустите:
```
./main
```
**Manual:**
Скачайте main.cpp:
```
git clone https://github.com/igoresaa/timetumbler
cd timetumbler
```
Для компиляции вам понадобиться C++ 23 и выше. Скомпилировать можно с помощью gcc:
```
g++ -std=c++23 main.cpp
```
Запустите:
```
./main
```
